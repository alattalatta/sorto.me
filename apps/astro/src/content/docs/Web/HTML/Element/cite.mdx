**HTML `<cite>` 요소**는 저작물을 가리키는 참조를 나타냅니다. 반드시 저작물의 제목을 포함해야 합니다. 적절한 맥락 아래에서는 제목을 축약해서 표기할 수도 있습니다.

```html hidden name=demo
<blockquote>
  <p>4월, 맑고 쌀쌀한 날이었다. 괘종시계가 13시를 알렸다.</p>
  <footer>
    조지 오웰,
    <cite><a href="https://books.google.co.kr/books?id=-sVdQ32rW0QC&dq=1984">1984</a></cite
    >, 정희성 옮김, 민음사(2012), p9.
  </footer>
</blockquote>
```

```css hidden name=demo
cite {
  font-style: normal;
}
cite::before {
  content: '『';
}
cite::after {
  content: '』';
}
blockquote {
  margin: 0;
}
```

# 사용 일람

## 창작물의 종류

`<cite>` 요소가 가리키는 "저작물"은 다음과 같습니다.

- 책
- 논문
- 에세이
- 시
- 악보
- 음악
- 대본
- 영화
- TV 쇼
- 게임
- 조각상
- 그림
- 연극
- 공연
- 오페라
- 뮤지컬
- 전시회
- 판례
- 컴퓨터 프로그램
- 웹사이트
- 웹 페이지
- 블로그 글과 댓글
- 게시판 글과 댓글
- 트윗
- Facebook 글
- 서면 또는 구두 성명
- 그 외 기타 등등

## 명세의 차이

`<cite>` 요소의 콘텐츠에 대해서는 W3C HTML 5와 WHATWG HTML Living Standard에 차이가 있습니다. W3C에서는 `<cite>` 요소에 저작물의 저작자 이름까지 포함할 수 있다고 명시합니다. 이와 정반대로, WHATWG는 어떠한 상황에서도 저작자 이름을 `<cite>` 안에 넣으면 안된다고 규정합니다.

```html variant=bad
<!-- 문서 제목 외의 것이 포함됨 -->
<cite>위키백과의 HTML 문서</cite>
```

```html variant=good
<!-- 문서 제목만 포함 -->
위키백과의 <cite>HTML</cite> 문서
```
