**HTML `<rt>` 요소**는 [`<ruby>`](./ruby)를 사용한 루비 표기법에서 위, 아래, 옆에 붙는 작은 글자인 루비 텍스트 나타냅니다. `<rt>`는 반드시 `<ruby>` 요소 안에 배치해야 합니다.

```html hidden name=demo
<ruby> 景 <rp>(</rp><rt>경</rt><rp>)</rp> 福 <rp>(</rp><rt>복</rt><rp>)</rp> 宮 <rp>(</rp><rt>궁</rt><rp>)</rp> </ruby>
```

```css hidden name=demo
ruby {
  font-size: 4em;
}
```
