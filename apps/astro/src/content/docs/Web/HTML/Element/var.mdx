**HTML `<var>` 요소**는 수학 표현식과 코드의 변수 이름을 나타냅니다.

```html hidden name=demo
<p>
  박스의 길이, 너비, 높이를 각각 <var>l</var>, <var>w</var>, <var>h</var>라고 할 때, 박스의 부피는 <var>l</var> ×
  <var>w</var> × <var>h</var>로 구할 수 있습니다.
</p>
```

```css hidden name=demo
var {
  font-family: serif;
  font-weight: 700;
}
```
