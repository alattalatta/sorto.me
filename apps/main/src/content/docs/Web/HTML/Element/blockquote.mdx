**HTML `<blockquote>` 요소**는 감싸고 있는 텍스트가 긴 인용문임을 나타냅니다. 보통 들여쓰기를 적용한 모습으로 그려집니다. 출처의 URL은 [`cite`](../Global_attributes#term-cite) 특성으로 제공할 수 있고, 출처 텍스트는 [`<cite>`](./cite) 요소로 나타낼 수 있습니다.

```html hidden name=demo
<figure>
  <blockquote cite="https://html.spec.whatwg.org/multipage/grouping-content.html#the-blockquote-element">
    <p>The blockquote element represents a section that is quoted from another source.</p>
  </blockquote>
  <figcaption>—WHATWG, <cite>HTML Standard</cite></figcaption>
</figure>
```

```css hidden name=demo
blockquote {
  border-left: 4px solid darkturquoise;
  margin: 0 0 10px;
  padding: 2px 30px;
}
p {
  margin: 0;
}
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

- : `cite`
  - 출처 문서나 메시지를 가리키는 URL입니다. 인용문의 맥락 혹은 출처 정보를 가리킬 용도입니다.

# 사용 일람

- 짧은 인용문을 별도의 블록 없이 인라인으로 포함하려면 `<q>` 요소를 사용하세요.

# 명세
