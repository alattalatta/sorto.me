**HTML `<body>` 요소**는 HTML 문서의 콘텐츠를 나타냅니다. 하나의 문서에는 하나의 `<body>`만 존재할 수 있습니다.

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

- : `onafterprint`
  - 사용자가 문서를 출력한 후 호출할 함수입니다.
- : `onbeforeprint`
  - 사용자가 문서를 출력하기 전에 호출할 함수입니다.
- : `onbeforeunload`
  - 문서를 해제(unload)하기 전에 호출할 함수입니다.
- : `onblur`
  - 문서가 포커스를 잃었을 때 호출할 함수입니다.
- : `onerror`
  - 문서를 정상적으로 불러오지 못했을 때 호출할 함수입니다.
- : `onfocus`
  - 문서가 포커스를 얻었을 때 호출할 함수입니다.
- : `onhashchange`
  - 문서의 현재 주소 중 프래그먼트 식별자(해시(`#`)로 시작하는 부분)가 바뀔 때 호출할 함수입니다.
- : `onlanguagechange`
  - 사용자의 선호 언어가 바뀔 때 호출할 함수입니다.
- : `onload`
  - 문서를 다 불러왔을 때 호출할 함수입니다.
- : `onmessage`
  - 문서가 메시지를 받았을 때 호출할 함수입니다.
- : `onoffline`
  - 네트워크 연결이 실패했을 때 호출할 함수입니다.
- : `ononline`
  - 네트워크 연결이 복구됐을 때 호출할 함수입니다.
- : `onpopstate`
  - 사용자가 세션 기록을 탐색했을 때 호출할 함수입니다.
- : `onresize`
  - 문서의 크기가 바뀔 때 호출할 함수입니다.
- : `onstorage`
  - 스토리지 영역이 바뀔 때 호출할 함수입니다.
- : `onunload`
  - 문서가 해제(unload)되고 있을 때 호출할 함수입니다.

# 예제

```html
<!DOCTYPE html>
<html lang="ko">
  <head></head>
  <body>
    <h1>HTML Standard</h1>
  </body>
</html>
```
