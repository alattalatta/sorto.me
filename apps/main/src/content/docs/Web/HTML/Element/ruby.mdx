**HTML `<ruby>` 요소**는 기준 텍스트의 위, 아래, 옆에 작게 글자를 추가하는 글자를 나타내며, 주로 동아시아 문자의 발음을 표기할 때 사용합니다. 다른 텍스트를 추가할 때도 사용할 수는 있으나 흔히 쓰이지는 않습니다.

"루비"라는 이름은 활자 규격의 명칭에서 가져온 것입니다. 자세한 정보는 [위키백과의 루비 문자](https://ko.wikipedia.org/wiki/%EB%A3%A8%EB%B9%84_%EB%AC%B8%EC%9E%90)를 참고하세요.

```html hidden name=demo
<ruby> 景 <rt>경</rt> 福 <rt>복</rt> 宮 <rt>궁</rt> </ruby>
```

```css hidden name=demo
ruby {
  font-size: 4em;
}
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 예제

```html hidden name=char
<ruby> 韓 <rp>(</rp><rt>한</rt><rp>)</rp> 國 <rp>(</rp><rt>국</rt><rp>)</rp> 語 <rp>(</rp><rt>어</rt><rp>)</rp> </ruby>
```

```css editable=false name=char hidden
body {
  font-size: 1.5em;
}
```
