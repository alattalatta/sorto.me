CSS **@규칙**은 CSS의 동작 방식을 설정하는 명령문입니다. @규칙은 스타일 규칙과 다른 @규칙을 그룹으로 묶거나, 선택된 요소와 직접적인 관계는 없는 스타일 정보를 선언하거나, 가져오기와 이름공간 키워드 매핑등에 사용합니다.

# 구문

@규칙은 골뱅이표 `@` (U+0040 COMMERCIAL AT)으로 시작하고, 규칙 이름으로 이어집니다. 이후 다음 세미콜론 `;` (U+003B SEMICOLON) 또는 다음 CSS 블록을 만날 때까지 전부 @규칙의 일부로 해석합니다.
