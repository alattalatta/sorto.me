---
title: 'CSS 함수'
---

CSS **함수**는 특별한 데이터 처리나 계산을 수행한 후 CSS 속성에 사용할 수 있는 CSS 값을 반환하는 명령문입니다. CSS 함수는 일반 자료형보다 더 복잡한 데이터 유형을 나타내며, 반환 값 계산에 필요한 매개변수를 받을 수 있습니다.

# 구문

```css
selector {
  property: function([argument]? [, argument]!);
}
```

함수 구문은 함수의 이름으로 시작하고 여는 괄호 `(`로 이어집니다. 다음에는 함수에 전달하는 인수들이고, 마지막에 닫는 괄호 `)`가 옵니다.

함수는 여러 개의 매개변수를 받을 수 있고, CSS 속성 값과 비슷한 형식을 따릅니다. 어떤 함수들에서는 여러 인수를 쉼표로 구분하고, 다른 함수들에서는 공백으로 구분합니다.
