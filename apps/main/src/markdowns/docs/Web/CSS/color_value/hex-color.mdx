CSS **`<hex-color>`** 자료형은 빨강, 초록, 파랑과 투명도를 16진수 숫자로 표기해 [sRGB](https://ko.wikipedia.org/wiki/SRGB) 색을 나타내기 위한 표기법입니다.

[`<color>`](.)를 사용할 수 있는 모든 곳에는 `<hex-color>`를 사용할 수 있습니다.

# 구문

```css
/* 3개 값 구문 */
#RGB
/* 4개 값 구문 */
#RGBA
/* 6개 값 구문 */
#RRGGBB
/* 8개 값 구문 */
#RRGGBBAA
```

`<hex-color>` 자료형은 한 개의 `#`과, 그 뒤를 잇는 3개, 4개, 6개, 또는 8개의 16진수 숫자로 구성됩니다. 대소문자는 구분하지 않습니다.

- : `RR`
  - 색상의 빨강 성분을 나타내는 16진수 숫자로, `0`부터 `ff`(255) 사이의 값입니다. 한 글자만 지정한 경우, 즉 3개 또는 4개 값 구문의 `R`의 경우 같은 숫자를 복사해서 `RR`과 같이 취급합니다. 예를 들어 `1`은 `11`과 같습니다.
- : `GG`
  - 색상의 초록 성분을 나타내는 16진수 숫자로, `0`부터 `ff`(255) 사이의 값입니다. 한 글자만 지정한 경우, 즉 3개 또는 4개 값 구문의 `G`의 경우 같은 숫자를 복사해서 `G`와 같이 취급합니다. 예를 들어 `c`는 `cc`와 같습니다.
- : `BB`
  - 색상의 파랑 성분을 나타내는 16진수 숫자로, `0`부터 `ff`(255) 사이의 값입니다. 한 글자만 지정한 경우, 즉 3개 또는 4개 값 구문의 `B`의 경우 같은 숫자를 복사해서 `BB`와 같이 취급합니다. 예를 들어 `9`는 `99`와 같습니다.
- : `AA`
  - 색상의 투명도를 나타내는 알파 성분으로, `0`부터 `ff`(255) 사이의 값입니다. 한 글자만 지정한 경우, 즉 3개 또는 4개 값 구문의 `A`의 경우 같은 숫자를 복사해서 `AA`와 같이 취급합니다. 예를 들어 `e`는 `ee`와 같습니다. `0` 또는 `00`은 완전히 투명한 색이고, `f` 또는 `ff`는 완전히 불투명한 색입니다.
