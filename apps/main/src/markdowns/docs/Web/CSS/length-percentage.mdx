CSS **`<length-percentage>`** 자료형은 [`<length>`]와 [`<percentage>`] 모두 허용하는 값을 나타냅니다.

`<length-percentage>`를 값으로 받는 속성이라면 백분율이 길이로 계산됩니다. 따라서 [`calc()`]에서 백분율을 사용할 수 있습니다. 즉, 다음 세 [`width`] 모두 유효합니다.

```css-prop
width: 200px;
width: 20%;
width: calc(100% - 200px);
```
