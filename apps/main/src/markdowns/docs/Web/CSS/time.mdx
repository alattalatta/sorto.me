CSS **`<time>`** 자료형은 초 또는 밀리초 단위의 시간을 표현합니다. 애니메이션과 트랜지션에 관련된 속성들에서 사용합니다.

# 구문

`<time>` 구문은 [`<number>`](./number)와 그 뒤의 단위로 구성됩니다. 선택적으로 `+`와 `-` 부호를 앞에 붙일 수 있습니다. 다른 크기 단위와 마찬가지로, 수와 단위 리터럴 사이에 공백이 있어선 안됩니다.

> [note] 초와 밀리초 단위 모두에서 `0`은 같은 길이지만, [`<angle>`](./angle)이나 [`<length>`](./length)와 달리 `<time>`에서는 단위를 생략할 수 없습니다. 즉 `0`은 `0s`와 `0ms`를 나타낼 수 없으며 유효하지 않습니다.

## 단위

- : `s`
  - 시간을 초 단위로 나타냅니다.
- : `ms`
  - 시간을 밀리초 단위로 나타냅니다.

# 예제

## 유효한 값

```md variant=good
12s    양의 정수
-456ms 음의 정수
4.3ms  실수
14mS   단위는 대소문자를 구분하지 않음
+0s    양의 부호, 0, 단위
-0ms   음의 부호, 0, 단위
```

## 유효하지 않은 값

```md variant=bad
0    <time>에서는 단위 없는 0 사용 불가
12.0 단위가 없으므로 <time>이 아니라 <number>
7 ms 단위와 숫자 사이 공백
```
