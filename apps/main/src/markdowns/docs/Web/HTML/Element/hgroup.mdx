**HTML `<hgroup>` 요소**는 문서 구획의 다단 헤딩을 나타내며, 부제목 등을 나타내는 다수의 [`<p>`](./p) 요소를 하나의 [`<h1> - <h6>`]('./Heading_Elements') 요소와 함께 묶을 때 사용합니다.

```html hidden name=demo
<hgroup>
  <h1>HTML &lt;hgroup&gt;: 제목 그룹 요소</h1>
  <p>부제목을 나타내는 법</p>
</hgroup>
```

```css hidden name=demo
hgroup {
  border-left: 0.25rem solid maroon;
  padding-left: 2rem;
}

h1 {
  font-size: 2rem;
}
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

`<hgroup>` 요소를 사용하면 헤딩을 부제나 태그라인 따위의 부가적인 내용과 함께 묶을 수 있습니다. 이때 헤딩을 제외한 나머지는 `<hgroup>` 내의 [`<p>`](./p) 요소로 나타냅니다.

`<hgroup>` 자체는 문서의 아웃라인에 영향을 주지 않습니다. 아웃라인에 반영되는 건 그 안의 헤딩 요소입니다.

# 접근성

접근성 차원에서, 지금은 `<hgroup>`이 특별히 포함하는 의미가 없습니다. 브라우저 접근성 API에 노출되는 건 그 내부의 헤딩과 문단 요소들 뿐입니다.
