**HTML `<hr>` 요소**는 문단 레벨 요소의 주제 전환을 나타냅니다.

```html hidden name=demo
<p>§1: 파이트 클럽의 규칙 1조: 파이트 클럽에 대해 말하지 않는다.</p>
<hr />
<p>§2: 파이트 클럽의 규칙 2조: 항상 컵케이크를 가져올 것.</p>
```

<LiveExample height={180} lang="html" name="demo" />

역사적으로 `<hr>`은 가로줄로 렌더링했습니다. 그래픽 브라우저에서는 여전히 가로줄로 그리고 있지만, 이제 `<hr>`은 가로줄이라는 단순한 시각적 정의가 아닌, 주제의 전환이라는 구체적인 의미를 가지고 있습니다. 따라서 단순한 가로줄이 필요한 경우에는 CSS를 사용해야 합니다.
