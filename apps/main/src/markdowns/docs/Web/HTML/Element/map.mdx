**HTML `<map>` 요소**는 [`<area>`](./area) 요소와 함께 이미지 맵을 구성합니다.

{/* [todo] Demo */}

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

- : `name`
  - 이 이미지 맵을 참조할 때 사용할 수 있는 이름을 지정합니다. 필수 특성으로, 값이 비어있거나, 공백 문자를 포함하거나, 현재 문서의 다른 `<map>`과 중복된 값을 사용할 수 없습니다. [`id`](../Global_attributes#term-id) 특성이 존재하는 경우 `name`의 값을 동일하게 지정해야 합니다.
