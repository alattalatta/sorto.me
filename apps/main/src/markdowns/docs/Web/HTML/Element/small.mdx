**HTML `<small>` 요소**는 덧붙이는 글이나, 저작권과 법률 표기 등의 부가적인 텍스트를 나타냅니다. `<small>`이 포함하는 글씨의 크기가 시각적으로 작아야 할 필요는 없습니다. 다만 기본 스타일에서의 `<small>`은 자신의 콘텐츠를 한 사이즈 작은 글꼴(`small`에서 `x-small` 사이)로 표시합니다.

```html hidden name=demo
<article>
  <p>Docs는 웹 기술의 참고서를 제공합니다.</p>
  <hr />
  <p>
    <small>Docs 문서는 CC BY-SA 4.0 아래에서 자유롭게 이용할 수 있습니다.</small>
  </p>
  <article></article>
</article>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 예제

## 부가 텍스트

```html hidden name=option
<h1>진행 중인 이벤트</h1>
<ul>
  <li>오렌지 주스 1+1 <small>~2021.03.01</small></li>
  <li>떡.튀.순 콤보 출시 <small>~2021.01.01</small></li>
  <li>3만원 이상 구매 시 최대 5% 적립 <small>상시 진행</small></li>
</ul>
```
