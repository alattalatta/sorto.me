**HTML `<caption>` 요소**는 표에 설명 또는 제목을 추가합니다.

<HTMLDemo height={180}>

```html
<table>
  <caption>우리 매장 원산지 표기</caption>
  <thead>
    <tr>
      <th>종류</th>
      <th>품목</th>
      <th>원산지</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th rowspan="3">과일</th>
      <td>사과</td>
      <td rowspan="2">국산</td>
    </tr>
    <tr>
      <td>배</td>
    </tr>
    <tr>
      <td>파인애플</td>
      <td>수입산</td>
    </tr>
    <tr>
      <th>곡물</th>
      <td>쌀</td>
      <td>국산</td>
    </tr>
    <tr>
      <th>해산물</th>
      <td>고등어</td>
      <td>원양산</td>
    </tr>
  </tbody>
</table>
```

```css
th,
td {
  border: 1px solid #777;
  padding: 8px;
}

th {
  background: #efefef;
}
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

`<caption>` 요소를 사용할 경우, 반드시 [`<table>`](./table)의 첫 번째 자식 요소로 배치해야 합니다. 위치를 아래로 바꾸고자 하면 `<table>`에 CSS [`caption-side`](https://developer.mozilla.org/en-US/docs/Web/CSS/caption-side) 속성을 설정하세요.

`<table>`의 [`background-color`](https://developer.mozilla.org/en-US/docs/Web/CSS/background-color)는 `<caption>`을 포함하지 않습니다. `<caption>`에 따로 `background-color`를 설정하세요.
