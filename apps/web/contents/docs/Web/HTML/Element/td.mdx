**HTML `<td>` 요소**는 표의 한 칸을 정의합니다.

<HTMLDemo height={180}>

```html
<table>
  <caption>우리 매장 원산지 표기</caption>
  <thead>
    <tr>
      <th>종류</th>
      <th>품목</th>
      <th>원산지</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th rowspan="3">과일</th>
      <td>사과</td>
      <td rowspan="2">국산</td>
    </tr>
    <tr>
      <td>배</td>
    </tr>
    <tr>
      <td>파인애플</td>
      <td>수입산</td>
    </tr>
    <tr>
      <th>곡물</th>
      <td>쌀</td>
      <td>국산</td>
    </tr>
    <tr>
      <th>해산물</th>
      <td>고등어</td>
      <td>원양산</td>
    </tr>
  </tbody>
</table>
```

```css
th,
td {
  border: 1px solid #777;
  padding: 8px;
}

th {
  background: #efefef;
}
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

- : `colspan`
  - 이 칸이 차지해야 하는 열의 수입니다. 기본 값은 1입니다. 자연수가 아니거나 1000을 초과하는 값은 유효하지 않으며 기본 값(1)을 대신 사용합니다.
- : `headers`
  - 이 칸을 설명하는 [`<th>`](./th) 요소들의 [`id`](../Global_attributes#term-id)를 가리키는 쉼표 구분 리스트입니다.
- : `rowspan`
  - 이 칸이 차지해야 하는 행의 수입니다. 기본 값은 1입니다. 0을 지정할 경우 자신이 속한 테이블 구획([`<thead>`](./thead), [`<tbody>`](./tbody), [`<tfoot>`](./tfoot), 암시적으로 생성되는 경우도 포함)의 끝까지 확장됩니다. 65534를 초과하는 값을 지정하면 65534를 대신 사용합니다.
