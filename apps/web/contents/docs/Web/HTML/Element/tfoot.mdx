**HTML `<tfoot>` 요소**는 표에서 열의 푸터를 구성하는 행들을 묶습니다.

<HTMLDemo height={180}>

```html
<table>
  <thead>
    <tr>
      <th>항목</th>
      <th scope="col">지출</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">식빵</th>
      <td>3,000</td>
    </tr>
    <tr>
      <th scope="row">커피</th>
      <td>4,500</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th scope="row">합계</th>
      <td>7,500</td>
    </tr>
  </tfoot>
</table>
```

```css
table {
  border-collapse: collapse;
}

thead,
tfoot {
  background: #efefef;
}

th,
td {
  border: 1px solid #777;
  padding: 8px;
}
```

</HTMLDemo>
