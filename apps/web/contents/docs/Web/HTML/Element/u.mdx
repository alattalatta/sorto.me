**HTML `<u>` 요소**는 철자 오류 표시처럼, 텍스트로 표기하지는 않지만 명시적으로 렌더링되는 주석이 적용된 인라인 텍스트를 나타냅니다.

<HTMLDemo height={80}>

```html
<p>밥 왜 안먹어?</p>
<p>8시 전까지는 먹으면 <u>않됀데</u>.</p>
```

```css
u {
  text-decoration: red wavy underline;
}
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

과거의 `<u>` 요소는 밑줄(underline)을 그리기 위해 사용했으며, 요소의 이름도 **U**nderline에서 따온 것입니다. 지금도 대부분의 브라우저에서는 여전히 굵은 글씨로 렌더링 하지만, 이제는 단순히 밑줄을 그리기 위한 용도로 사용해서는 안됩니다. CSS[`text-decoration`](https://developer.mozilla.org/en-US/docs/Web/CSS/text-decoration)을 사용하세요.

많은 경우, `<u>` 요소보다는 다른 요소가 용도에 적합합니다.

- 키워드와 같은 텍스트의 강조는 [`<b>`](./b) 또는 [`<mark>`](./mark)를 사용하세요.
- 강세를 나타내려면 [`<em>`](./em)을 사용하세요.
- 중요한 부분은 [`<strong>`](./strong)을 사용하세요.
