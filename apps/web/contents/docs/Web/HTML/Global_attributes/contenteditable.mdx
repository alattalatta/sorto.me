HTML **`contenteditable`** [전역 특성](.)은 사용자가 요소를 편집할 수 있는지 지정하는 열거형 특성입니다.

<HTMLDemo height={180}>

```html
<blockquote contenteditable>
  <p>자신만의 명언을 적어보세요</p>
</blockquote>

<cite contenteditable>-- 이름을 적어주세요</cite>
```

```css
blockquote {
  background: #eee;
  border-radius: 5px;
  color: #000;
  margin: 0 0 16px;
  padding: 15px;
}

p {
  margin: 0;
}

blockquote p::before {
  content: '\201C';
}

blockquote p::after {
  content: '\201D';
}
```

</HTMLDemo>

가능한 값은 다음과 같습니다.

- `true` 또는 빈 문자열: 요소를 편집할 수 있습니다.
- `false`: 요소를 편집할 수 없습니다.

값 없이 `<label contenteditable>예제</label>`처럼 사용할 경우 빈 문자열 값을 지정한 것으로 취급해서, 요소를 편집할 수 있습니다.

`contenteditable` 특성을 명시하지 않았거나 그 값이 유효하지 않으면, 편집 가능 여부는 부모 요소에게서 상속합니다.

`contenteditable`이 허용하는 값에 `true`와 `false`가 존재하긴 하지만, `contenteditable`은 _불리언_ 특성이 아니고 _열거형_ 특성임에 주의하세요.
