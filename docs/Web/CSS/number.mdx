**CSS `<number>` 자료형**은 숫자, 즉 정수 또는 실수를 표현합니다.

# 구문

`<number>` 구문은 [`<integer>`](./integer)의 구문을 확장합니다. 소수점 이하 값은 `.` 뒤로 한 개 이상의 10진수 숫자를 붙여 표현하며, 정수를 앞에 붙일 수 있습니다. `<number>`에는 단위가 없습니다.

# 보간

애니메이션에서 `<number>`의 값은 유동소수점 실수를 사용해 보간합니다. 보간의 속도는 애니메이션과 연결된 [타이밍 함수](https://developer.mozilla.org/en-US/docs/Web/CSS/easing-function)가 결정합니다.

# 예제

## 유효한 값

```md variant=good
12          <integer>는 <number>
4.01        양의 실수
-456.8      음의 실수
0.0         0
+0.0        0, 양의 부호
-0.0        0, 음의 부호
.60         앞의 0 없는 소수점 이하 값
10e3        과학적 표기법
-3.4e-2     복잡한 과학적 표기법
```

## 유효하지 않은 값

```md variant=bad
12.         소수점 뒤에 최소 하나의 숫자가 존재해야 함
+-12.2      하나의 +/-만 허용
12.1.1      하나의 소수점만 허용
```
