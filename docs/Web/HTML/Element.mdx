HTML 요소 참고서는 태그를 사용해 생성할 수 있는 HTML 요소의 목록을 제공합니다.

# 메인 루트

- [`<html>`](./Element/html)

# 문서 메타데이터

메타데이터는 페이지의 각종 정보로서 스타일, 스크립트, (검색 엔진, 브라우저 등) 소프트웨어가 페이지를 렌더링 할 때 사용할 수 있는 데이터 등을 담습니다. 스타일과 스크립트 메타데이터는 페이지 안에 정의할 수도 있고 다른 파일을 가리키는 링크일 수도 있습니다.

- [`<base>`](./Element/base)
- [`<head>`](./Element/head)
- [`<link>`](./Element/link)
- [`<meta>`](./Element/meta)
- [`<style>`](./Element/style)
- [`<title>`](./Element/title)

# 구획 루트

- [`<body>`](./Element/body)

# 콘텐츠 구획

콘텐츠 구획 요소를 사용해서 문서의 콘텐츠를 논리적 구획으로 체계화할 수 있습니다. 헤더, 푸터, 탐색 영역 등으로 페이지의 큰 틀을 잡고, 헤딩 요소를 사용해 각 구획에 이름을 붙이세요.

- [`<address>`](./Element/address)
- [`<article>`](./Element/article)
- [`<aside>`](./Element/aside)
- [`<footer>`](./Element/footer)
- [`<header>`](./Element/header)
- [`<h1>`, `<h2>`, `<h3>`](./Element/Heading_Elements), ...
- [`<hgroup>`](./Element/hgroup)
- [`<main>`](./Element/main)
- [`<nav>`](./Element/nav)
- [`<section>`](./Element/section)

# 텍스트 콘텐츠

- [`<blockquote>`](./Element/blockquote)
- [`<dd>`](./Element/dd)
- [`<div>`](./Element/div)
- [`<dl>`](./Element/dl)
- [`<dt>`](./Element/dt)

# 이미지와 멀티미디어

HTML은 사진, 오디오, 비디오 등 다양한 멀티미디어 리소스를 지원합니다.

- `<area>`
- `<audio>`
- `<img>`
- `<map>`
- `<track>`
- `<video>`

# 콘텐츠 임베딩

일반적인 멀티미디어 콘텐츠 외에도 다양한 종류의 다른 콘텐츠 역시 포함할 수 있습니다.

- `<embed>`
- `<iframe>`
- `<object>`
- `<param>`
- `<picture>`
- `<poratl>`
- `<source>`
- `<svg>`

# 스크립트

동적인 콘텐츠와 웹 어플리케이션을 위해, HTML은 스크립트 언어, 그 중에서도 주로 JavaScript를 지원합니다. 다음 요소에서 스크립트를 지원합니다.

- `<canvas>`
- `<noscript>`
- `<script>`

# 편집 영역 표시

편집 영역 표시 요소를 사용하면 텍스트에서 수정한 부분을 표시할 수 있습니다.

- [`<del>`](./Element/del)
- [`<ins>`](./Element/ins)

- `<td>`
- `<tfoot>`
- `<th>`

- `<tr>`

# 양식

HTML은 여러가지 입력 가능한 요소를 제공합니다. 이 요소들을 서로 조합하면 사용자가 내용을 채운 후 웹사이트나 어플리케이션에 제출할 수 있습니다.

- `<button>`
- `<datalist>`
- `<fieldset>`
- `<form>`
- `<input>`
- `<label>`
- `<legend>`
- `<meter>`
- `<optgroup>`
- `<option>`
- `<output>`
- `<progress>`
- `<select>`
- `<textarea>`
