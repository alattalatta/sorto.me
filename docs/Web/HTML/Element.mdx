# 문서 메타데이터

메타데이터는 페이지의 각종 정보로서 스타일, 스크립트, (검색 엔진, 브라우저 등) 소프트웨어가 페이지를 렌더링 할 때 사용할 수 있는 데이터 등을 담습니다. 스타일과 스크립트 메타데이터는 페이지 안에 정의할 수도 있고 다른 파일을 가리키는 링크일 수도 있습니다.

- [`<head>`](./Element/head)
- `<link>`
- [`<meta>`](./Element/meta)
- `<style>`
- [`<title>`](./Element/title)
