**HTML `<aside>` 요소**는 문서의 주요 내용에 간접적으로만 연관된 부분을 나타내며, 주로 사이드바와 콜아웃 박스에 사용합니다.

```html
<article>
  <p>
    HTML &lt;aside&gt; 요소는 문서의 주요 내용에 간접적으로만 연관된 부분을 나타내며, 주로 사이드바와 콜아웃 박스에
    사용합니다.
  </p>
  <aside>
    <p>콜아웃은 이렇게 중간에 추가 설명을 넣는 영역을 의미합니다.</p>
  </aside>
  <h2>사용 일람</h2>
  <!-- ... -->
</article>
```

# 특성

[전역 특성](https://developer.mozilla.org/ko/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

- 텍스트의 중간에 괄호를 사용해 설명을 추가하는 경우에는 `<aside>`를 사용하지 않아야 합니다. 주제와 직접적인 연관이 없더라도 문서 메인 흐름의 일부로 취급하기 때문입니다.

# 명세

[HTML Living Standard](https://html.spec.whatwg.org/multipage/semantics.html#the-aside-element)
