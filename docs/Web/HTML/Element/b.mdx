**HTML `<b>` 요소**는 특별한 중요도는 가지지 않으면서 독자의 주의를 끌어야 하는 콘텐츠를 나타냅니다.

<HTMLDemo height={120}>

```html
<p>
  태양과 비슷한 질량의 항성이 수명을 다 하면 <b>백색왜성</b>으로 변화합니다. 반면, 태양 질량의 약 세 배 정도의 항성은
  강력한 중력으로 인해 <b>중성자별</b>이 되고, 그보다 더 큰 초대형 항성은 중력붕괴를 이기지 못해 <b>블랙홀</b>이 됩니다.
</p>
```

```css hidden
p {
  line-height: 1.7;
}
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

과거에 `<b>` 요소는 "굵은 글꼴"(boldface) 요소였으며, 요소 이름인 `b`도 **B**oldface에서 따온 것입니다. 그래서 지금도 대부분의 브라우저에서는 여전히 굵은 글씨로 렌더링 하지만, 이제는 단순히 글씨 굵기를 조절하기 위한 용도로 사용해서는 안됩니다. 대신 CSS [`font-weight`](/docs/Web/CSS/font-weight)를 사용하세요. 또는, 중요한 내용임을 표시하기 위함이라면 [`<strong>`](./strong) 요소를 대신 사용하세요.

- `<b>` 요소는 개요의 키워드나 리뷰의 제품 이름처럼, 시선은 끌어야 하지만 특별한 중요도는 갖지 않는 곳에 사용하세요.
- `<b>` 요소와 `<strong>`, [`<em>`](./em), [`<mark>`](./mark) 요소를 구별하세요. `<strong>`은 "중요한" 내용을, `<em>`은 강세를, `<mark>`는 관련성을 나타낼 때 사용합니다. 다른 요소가 적합하지 않을 때만 `<b>`를 사용하세요.
- 마찬가지로, 제목과 헤딩을 `<b>`로 나타내면 안됩니다. [헤딩 요소](./Heading_Elements)를 사용하세요.
