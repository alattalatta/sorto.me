**HTML `<body>` 요소**는 HTML 문서의 콘텐츠를 나타냅니다. 하나의 문서에는 하나의 `<body>`만 존재할 수 있습니다.

```html
<!DOCTYPE html>
<html lang="ko">
  <head></head>
  <body>
    <h1>HTML Living Standard</h1>
  </body>
</html>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

<dl>
  <Term>onafterprint</Term>
  <dd>사용자가 문서를 출력한 후 호출할 함수입니다.</dd>
  <Term>onbeforeprint</Term>
  <dd>사용자가 문서를 출력하기 전에 호출할 함수입니다.</dd>
  <Term>onbeforeunload</Term>
  <dd>문서를 해제(unload)하기 전에 호출할 함수입니다.</dd>
  <Term>onblur</Term>
  <dd>문서가 포커스를 잃었을 때 호출할 함수입니다.</dd>
  <Term>onerror</Term>
  <dd>문서를 정상적으로 불러오지 못했을 때 호출할 함수입니다.</dd>
  <Term>onfocus</Term>
  <dd>문서가 포커스를 얻었을 때 호출할 함수입니다.</dd>
  <Term>onhashchange</Term>
  <dd>
    문서의 현재 주소 중 프래그먼트 식별자(해시(<code>#</code>)로 시작하는 부분)가 바뀔 때 호출할 함수입니다.
  </dd>
  <Term experimental={true}>onlanguagechange</Term>
  <dd>사용자의 선호 언어가 바뀔 때 호출할 함수입니다.</dd>
  <Term>onload</Term>
  <dd>문서를 다 불러왔을 때 호출할 함수입니다.</dd>
  <Term>onmessage</Term>
  <dd>문서가 메시지를 받았을 때 호출할 함수입니다.</dd>
  <Term>onoffline</Term>
  <dd>네트워크 연결이 실패했을 때 호출할 함수입니다.</dd>
  <Term>ononline</Term>
  <dd>네트워크 연결이 복구됐을 때 호출할 함수입니다.</dd>
  <Term>onpopstate</Term>
  <dd>사용자가 세션 기록을 탐색했을 때 호출할 함수입니다.</dd>
  <Term>onredo</Term>
  <dd>사용자가 실행 기록에서 앞으로 이동했을 때 호출할 함수입니다.</dd>
  <Term>onresize</Term>
  <dd>문서의 크기가 바뀔 때 호출할 함수입니다.</dd>
  <Term>onstorage</Term>
  <dd>스토리지 영역이 바뀔 때 호출할 함수입니다.</dd>
  <Term>onundo</Term>
  <dd>사용자가 실행 기록에서 뒤로 이동했을 때 호출할 함수입니다.</dd>
  <Term>onunload</Term>
  <dd>문서가 해제(unload)되고 있을 때 호출할 함수입니다.</dd>
</dl>
