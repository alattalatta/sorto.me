**HTML `<figcaption>` 요소**는 부모 [`<figure>`](./figure) 요소의 콘텐츠에 대한 설명 또는 범례를 제공합니다.

<Demo height={260} />

```css name=demo sub
figure {
  border: 1px solid #000;
  max-width: 320px;
  text-align: center;
  padding: 8px;
}
img {
  width: 100%;
}
figcaption {
  font-size: 12px;
}
```

```html name=demo
<figure>
  <img src="/images/docs/plumbing.jpg" alt="복잡한 배관 사진." />
  <figcaption>모범 사례</figcaption>
</figure>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.
