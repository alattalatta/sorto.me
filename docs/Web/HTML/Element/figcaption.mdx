**HTML `<figcaption>` 요소**는 부모 [`<figure>`](./figure) 요소의 콘텐츠에 대한 설명 또는 범례를 제공합니다.

```html
<figure>
  <img src="/media/example/elephant.jpg" alt="진흙 목욕을 하는 코끼리">
  <figcaption>코끼리의 생태</figcaption>
</figure>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 명세

[HTML Living Standard](https://html.spec.whatwg.org/multipage/semantics.html#the-figcaption-element)
