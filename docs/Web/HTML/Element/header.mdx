**HTML `<header>` 요소**는 소개와 탐색 콘텐츠를 나타냅니다. 보통 헤딩 요소([`<h1> - <h6>`](./Heading_Elements) 또는 [`<hgroup>`](./hgroup))를 포함하지만, 그 외에 로고, 검색 창, 작성자 이름 등 여러가지를 포함할 수 있습니다.

<Demo height={120} />

```html name=demo
<header>
  <h1>HTML Living Standard</h1>
</header>
<main>
  <p><strong>HTML Living Standard</strong>는 WHATWG에서 관리하는 HTML 표준 명세입니다.</p>
</main>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 명세

[HTML Living Standard](https://html.spec.whatwg.org/multipage/sections.html#the-header-element)

# 브라우저 호환성

<BrowserCompat>html.elements.header</BrowserCompat>
