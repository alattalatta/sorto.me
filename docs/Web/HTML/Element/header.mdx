**HTML `<header>` 요소**는 소개와 탐색 콘텐츠를 나타냅니다. 보통 헤딩 요소(`<h1> - <h6>` 또는 `<hgroup>`)를 포함하지만, 그 외에 로고, 검색 창, 작성자 이름 등 여러가지를 포함할 수 있습니다.

```html
<header>
  <img src="/images/html-spec.jpg" alt="" />
  <p>HTML Living Standard</p>
  <h1>&lt;footer&gt; 요소</h1>
</header>
```

# 특성

[전역 특성](https://developer.mozilla.org/ko/docs/Web/HTML/Global_attributes)만 포함합니다.

# 명세

[HTML Living Standard](https://html.spec.whatwg.org/multipage/sections.html#the-header-element)
