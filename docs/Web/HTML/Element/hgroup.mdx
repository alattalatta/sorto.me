**HTML `<hgroup>` 요소**는 문서 구획의 다단 헤딩을 나타내며, 다수의 [`<h1> - <h6>`]('./Heading_Elements') 요소를 묶을 때 사용합니다.

<Demo height={120} />

```html name=demo
<hgroup>
  <h1>HTML Living Standard</h1>
  <h2>&lt;hgroup&gt; 요소</h2>
</hgroup>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

<Callout color="warn" label="참고">
  <code>&lt;hgroup&gt;</code> 요소는{' '}
  <Anchor href="https://html.spec.whatwg.org/multipage/sections.html#outline">HTML 아웃라인 알고리즘</Anchor>에서 하위
  헤딩 요소가 불필요한 구획을 생성하지 않도록, 다단 헤딩을 하나의 논리 헤딩으로 묶을 때 사용하는 요소입니다. 그러나
  아웃라인 알고리즘을 구현하는 브라우저는 하나도 없습니다.
</Callout>

- `<hgroup>` 요소의 헤딩 단계는 자신에게 속한 헤딩들의 단계 중 제일 높은 단계를 따라갑니다.

# 명세

[HTML Living Standard](https://html.spec.whatwg.org/multipage/semantics.html#the-hgroup-element)
