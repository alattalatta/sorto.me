title: 'HTML <hr>: 주제 전환 요소'
excerpt: HTML <hr> (가로줄, horizontal rule) 요소는 문단 레벨 요소의 주제 전환을 나타냅니다.
---

**HTML `<figure>` 요소**의 주제 전환을 나타냅니다.

<Demo height={120} />

```html name=demo
<p>§1: 파이트 클럽의 규칙 1조: 파이트 클럽에 대해 말하지 않는다.</p>
<hr />
<p>§2: 파이트 클럽의 규칙 2조: 항상 컵케이크를 가져올 것.</p>
```

역사적으로 `<hr>`은 가로줄로 표현했습니다. 그래픽 브라우저에서는 여전히 가로줄로 표현하고 있지만, 이제 명세 상의 정의는 시각적 표현이 아닌, 주제 전환이라는 별도의 의미를 가지고 있습니다. 따라서 단순한 가로줄이 필요한 것이면 CSS를 사용해야 합니다.
