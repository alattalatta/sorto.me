**HTML `<li>` 요소**는 목록의 항목을 나타냅니다. 반드시 정렬 목록([`<ol>`](./ol)) 또는 비정렬 목록([`<ul>`](./ul)) 아래에 위치해야 합니다. 보통 비정렬 목록에서는 각각의 항목 앞에 구분점이 붙고, 정렬 목록에서는 숫자나 로마자/영문자 등 오름차순 카운터가 붙습니다.

```html
<p>아폴로 13호의 승무원:<p>
<ul>
  <li>짐 러블</li>
  <li>잭 스와이거트</li>
  <li>프레드 헤이즈</li>
</ul>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

<dl>
  <Term>value</Term>
  <dd>
    <Anchor href="./ol"><code>{'<ol>'}</code></Anchor> 요소 아래에서의 현재 순서를 재정의하는 정수형 특성입니다. 리스트가 로마 숫자나 영문자를 쓰더라도 항상 숫자 값만 지정해야 합니다. 이후의 목록 항목은 이 값을 기준으로 시작합니다. 비정렬 목록(<Anchor href="./ul"><code>{'<ul>'}</code></Anchor>) 아래에서는 아무런 영향도 없습니다.
  </dd>
</dl>

# 예제

## 중간 숫자부터 시작하는 정렬 목록

```html
<ol type="I">
  <li value="3">셋</li>
  <li>넷</li>
  <li>다섯</li>
</ol>
```
