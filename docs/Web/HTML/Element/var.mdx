**HTML `<var>` 요소**는 수학 표현식과 프로그래밍에서 변수의 이름을 나타냅니다.

<HTMLDemo height={120}>

```html
<p>
  박스의 길이, 너비, 높이를 각각 <var>l</var>, <var>w</var>, <var>h</var>라고 할 때,
  박스의 부피는 <var>l</var> × <var>w</var> × <var>h</var>로 구할 수 있습니다.
</p>
```

```css
var {
  font-family: serif;
  font-weight: 700;
}
```

</HTMLDemo>
