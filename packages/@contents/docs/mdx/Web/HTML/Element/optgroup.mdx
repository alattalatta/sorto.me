**HTML `<optgroup>` 요소**는 [`<select>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select) 요소 내의 선택지를 그룹으로 묶습니다.

<HTMLDemo height={40}>

```html
<label for="dino">공룡 선택:</label>
<select id="dino">
  <optgroup label="백악기">
    <option>티라노사우루스</option>
    <option>벨로키랍토르</option>
  </optgroup>
  <optgroup label="쥐라기">
    <option>브라키오사우루스</option>
    <option>스테고사우루스</option>
  </optgroup>
</select>
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

- : `disabled`
  - 그룹에 속한 모든 선택지를 고를 수 없도록 하는 불리언 특성입니다.
- : `label`
  - 선택지 그룹의 이름으로 사용자에게 보여줄 값입니다. 반드시 지정해야 합니다.

# 사용 일람

`<optgroup>` 요소는 중첩할 수 없습니다.

# 명세
