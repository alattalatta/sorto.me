**HTML `<br>` 요소**는 텍스트에 줄바꿈(캐리지 리턴)을 추가합니다. 시와 주소처럼 줄바꿈이 중요한 경우에 사용할 수 있습니다.

<HTMLDemo height={180}>

```html
<p>
  별 하나에 추억과<br />
  별 하나에 사랑과<br />
  별 하나에 쓸쓸함과<br />
  별 하나에 동경과<br />
  별 하나에 시와<br />
  별 하나에 어머니, 어머니,
</p>
```

```css
p {
  line-height: 1.7;
}
```

</HTMLDemo>

`<br>` 요소를 추가한 곳에서 줄이 바뀌어서, `<br>` 뒤의 텍스트는 다음 줄에서 시작하는 것을 볼 수 있습니다.

<Callout.Root color="warn" label="참고">
  `<br>`을 사용해서 문단을 나누지 마세요. 문단은 [`<p>`](./p) 요소로 감싸서 나타내야 합니다. 문단 사이 간격은 CSS [`margin`](https://developer.mozilla.org/ko/docs/Web/CSS/margin) 속성으로 설정할 수 있습니다.
</Callout.Root>
