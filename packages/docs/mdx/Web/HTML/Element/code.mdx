**HTML `<code>` 요소**는 자신의 짧은 컴퓨터 코드 조각을 나타냅니다. 기본 스타일에서 `<code>`의 텍스트는 사용자 에이전트의 기본 고정폭 글꼴을 사용해 표시합니다.

<HTMLDemo height={70}>

```html
<p>
  배열의 <code>push()</code> 메서드는 배열의 맨 마지막에 하나 이상의 요소를 추가하고, 배열의 새로운 길이를 반환합니다.
</p>
```

```css
code {
  background: #eee;
  color: crimson;
  display: inline-block;
  padding: 2px 4px;
}
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

코드 여러 줄을 표현하려면 `<code>`를 [`<pre>`](./pre) 요소 안에 배치하세요. `<code>` 요소 자체로는 한 줄의 코드만 표현할 수 있습니다.

# 명세
