**HTML `<data>` 요소**는 자신의 콘텐츠를 기계가 판독할 수 있는 형태의 데이터와 연결합니다. 해당 데이터가 시간과 날짜라면 `<time>` 요소를 사용하세요.

<HTMLDemo height={120}>

```html
<p>가능한 객실:</p>
<ul>
  <li><data value="U3041">비즈니스</data></li>
  <li><data value="U7041">스위트</data></li>
  <li><data value="R81">로열 스위트</data></li>
</ul>
```

</HTMLDemo>

위의 예제는 사람이 보게 되는 각 객실의 이름을 기계가 판독할 때 사용할 객실 ID와 연결하고 있습니다.

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

<dl>
  <Term>value</Term>
  <dd>콘텐츠의 기계 판독 가능한 형태를 지정합니다.</dd>
</dl>
