**HTML `<hgroup>` 요소**는 문서 구획의 다단 헤딩을 나타내며, 다수의 [`<h1> - <h6>`]('./Heading_Elements') 요소를 묶을 때 사용합니다.

<HTMLDemo height={120}>

```html
<hgroup>
  <h1>HTML Living Standard</h1>
  <h2>&lt;hgroup&gt; 요소</h2>
</hgroup>
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

> [note] `<hgroup>` 요소는 [HTML 아웃라인 알고리즘](https://html.spec.whatwg.org/multipage/sections.html#outline)에서 하위 헤딩 요소가 불필요한 구획을 생성하는 일을 방지하기 위해, 다단 헤딩을 하나의 논리 헤딩으로 묶을 때 사용하는 요소입니다. 그러나 아웃라인 알고리즘을 구현하는 브라우저는 하나도 없습니다.

- `<hgroup>` 요소의 헤딩 단계는 자신에게 속한 헤딩들의 단계 중 제일 높은 단계를 따라갑니다.

# 명세
