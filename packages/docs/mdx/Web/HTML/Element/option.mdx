---

**HTML `<option>` 요소**는 [`<select>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select), [`<optgroup>`](./optgroup), [`<datalist>`](./datalist) 요소 내의 선택지를 나타냅니다.

<HTMLDemo height={40}>

```html
<label for="dino">공룡 선택:</label>
<select id="dino">
  <option value="">--선택하세요--</option>
  <option value="velociraptor">벨로키랍토르</option>
  <option value="brachiosaurus">브라키오사우루스</option>
  <option value="stegosaurus">스테고사우루스</option>
  <option value="tyrannosaurus">티라노사우루스</option>
</select>
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

<dl>
<Term>disabled</Term>
<dd>

이 선택지를 고를 수 없도록 하는 불리언 특성입니다. [`<optgroup>`](./optgroup)에 속한 경우, 이 특성 없이도 `<optgroup>`의 [`disabled`](./optgroup#term-disabled) 특성으로 인해 비활성화될 수 있습니다.

</dd>

<Term>label</Term>
<dd>사용자에게 보여줄 선택지의 이름입니다. 따로 지정하지 않을 경우 요소의 텍스트 내용을 보여줍니다.</dd>

<Term>selected</Term>
<dd>

처음에 이 선택지를 선택한 상태로 지정하는 불리언 특성입니다. [`multiple`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select#attr-multiple) 특성을 지정하지 않은 [`<select>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select) 요소에 속하는 선택지들의 경우 하나의 선택지만 `selected` 특성을 가질 수 있습니다.

</dd>

<Term>value</Term>
<dd>이 선택지를 선택한 상태로 양식 제출 시 실제로 전송할 값입니다. 따로 지정하지 않을 경우 요소의 텍스트 내용을 전송합니다.</dd>
</dl>
