**HTML `<sup>` 요소**는 활자 관습상의 이유로 위 첨자를 사용해야 하는 인라인 텍스트를 지정합니다.

<HTMLDemo height={40}>

```html
<p>2<sup>3</sup>은 2를 세 번 곱하라는 것이니 8입니다.</p>
```

</HTMLDemo>

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

`<sup>` 요소는 위 첨자로 작성하지 않으면 의미가 바뀌는 텍스트에만 사용해야 하며, 작은 글자를 위로 배치하는 시각적 효과를 위해 사용해서는 안됩니다. 예컨대, 수학에서 제곱하는 횟수를 나타내는 지수는 위 첨자로 표기해야 합니다. (12<sup>2</sup> vs. 122) 이런 경우에는 `<sup>`를 사용할 수 있습니다.

그 외에도 각주 표기에 위 첨자를 사용할 수도 있습니다. 아래의 예제를 참고하세요.

# 예제

## 각주 표기

각주 번호는 위 첨자로도 나타낼 수 있고, 아래 첨자로도 나타낼 수 있습니다. 위 첨자를 사용할 경우의 예제입니다.

```html name=footnote
<p>JavaScript는 1995년, NetScape<sup>1</sup>의 프로그래머였던 Brandan Eich가 창시한 스크립팅 언어<sup>2</sup>입니다.</p>
```
