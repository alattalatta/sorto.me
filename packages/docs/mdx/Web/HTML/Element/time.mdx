**HTML `<time>` 요소**는 특정 시간을 나타냅니다. 기계가 판독 가능한 형태의 시간 값을 [`datetime`](#term-datetime) 특성에 지정하면 검색 결과를 개선하거나, 알림 등의 시간 기반 기능을 구현할 때 사용할 수 있습니다.

<HTMLDemo height={100}>

```html
<p>이번 10주년 기념식은 오는 <time datetime="2025-07-06">7월 6일</time>에 중앙회관에서 열립니다.</p>
<p>
  <time datetime="20:00">오후 6시</time>부터 <time datetime="PT2H30M">2시간 30분</time>
  동안 지난 시간을 되돌아봅시다.
</p>
```

```css
time {
  font-weight: 700;
}
```

</HTMLDemo>

`<time>` 요소는 다음과 같은 시간을 표현할 수 있습니다.

- 24시간 표기 시계의 시각.
- 정확한 [그레고리력](https://ko.wikipedia.org/wiki/%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EB%A0%A5) 날짜. (시간대 정보 포함 가능)
- [시간](https://www.w3.org/TR/2014/REC-html5-20141028/infrastructure.html#valid-duration-string). (30초, 4시간 등)

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)을 포함합니다.

- : `datetime`
  - 요소가 표현할 시간 값. 아래의 사용 일람에서 설명하는 형식 중 하나여야 합니다.

# 사용 일람

`<time>` 요소는 기계가 판독할 수 있는 형태로 날짜와 시각 등을 표현할 때 사용합니다. 예를 들어 티켓 예매 이메일의 본문에, 사람에게 친숙한 표현법을 사용하면서도 기계가 읽기 쉬운 정확한 시간을 포함해서 사용자 에이전트의 일정을 자동 등록 기능에 도움을 줄 수 있습니다.

복잡한 계산상 문제로 인해, 그레고리력 도입 이전의 시각을 나타낼 때 사용해서는 안됩니다.

사람이 볼 시간 값은 요소 내부의 콘텐츠, 기계가 읽을 시간 값은 `datetime` 특성의 값입니다. 특성 값은 아래의 적절한 형식 중 하나여야 합니다. 또한 `datetime` 특성이 없는 경우 텍스트를 제외한 어떠한 자식도 사용해서는 안되며, 이 때는 텍스트 콘텐츠를 `datetime` 특성 대신 사용합니다.

## 유효한 시간 형식

- : 유효한 연도 문자열
  - `2021`
- : 유효한 연-월 문자열
  - `2021-09`
- : 유효한 연-월-일 문자열
  - `2021-09-12`
- : 유효한 월-일 문자열
  - `09-12`
- : 유효한 연-주차 문자열
  - `2021-W30`
- : 유효한 시각 문자열
  - `14:54`
  - `14:54:39`
  - `14:54:39.519`
- : 유효한 현지 시각 문자열
  - `2021-09-12T14:54:39.519`
  - `2021-09-12 14:54:39.519`
- : 유효한 국제 시각 문자열
  - `2021-09-12T14:54:39.519Z`
  - `2021-09-12T14:54:39.519+0900`
  - `2021-09-12T14:54:39.519+09:00`
  - `2021-09-12 14:54:39.519Z`
  - `2021-09-12 14:54:39.519+0900`
  - `2021-09-12 14:54:39.519+09:00`
- : 유효한 시간 문자열
  - `PT4H18M3S`
