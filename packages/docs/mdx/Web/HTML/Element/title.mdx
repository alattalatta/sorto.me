**HTML `<title>` 요소**는 브라우저의 제목 바와 페이지 탭에 보이는 문서 제목을 정의합니다. 안의 태그는 모두 무시하고 텍스트만 사용합니다.

```html
<title>HTML &lt;title&gt; 요소 - Sorto.me Docs</title>
```

# 특성

[전역 특성](/docs/Web/HTML/Global_attributes)만 포함합니다.

# 사용 일람

`<title>` 요소는 항상 [`<head>`](./head) 안에 위치해야 합니다.

## 페이지 제목과 SEO

페이지 제목은 검색 엔진 최적화([SEO](https://developer.mozilla.org/ko/docs/Glossary/SEO))에 큰 영향을 줍니다. 보통 짧고 일반적인 제목보단 길고 주제가 잘 나타나는 제목이 더 좋은 결과를 나타냅니다. 제목은 검색 엔진이 결과를 정렬할 때 사용하는 구성 요소 중 하나이며, 잠재적 사용자가 검색 결과를 훑어볼 때 끌어당길 수 있는 첫 수단입니다.

다음은 좋은 제목을 작성할 때 참고할 수 있는 몇 가지 지침입니다.

- 단어 한두 개로만 제목을 구성하는 걸 피하세요. 콘텐츠를 설명하는 문장을 넣거나, 사전과 참고 문서의 경우 용어와 정의를 짝지어 사용하세요.
- 검색 결과는 일반적으로 페이지 제목의 첫 55~60글자만 노출합니다. 이후의 텍스트는 보이지 않을 수도 있으므로 제목이 너무 길어지지 않도록 제한하세요. 제목이 반드시 길어져야 하는 경우, 중요한 부분을 앞에 배치해서 뒷부분이 없어지더라도 내용을 알아보지 못하는 상황을 방지하세요.
- "키워드 뭉치" 스타일의 제목은 피하세요. 제목이 단순한 단어 나열에 불과하다면 검색 알고리즘이 결과에서 순서를 내리곤 합니다.
- 하나의 웹 사이트 내에서는 제목이 최대한 중복되지 않도록 하세요. 중복이나 유사한 제목의 웹 페이지는 검색 결과의 정확도를 떨어트리는 요인입니다.
